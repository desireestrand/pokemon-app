# Pokémon Application

A simple React application that fetches data from PokéAPI and allows the user to view information about the first 151 Pokémon.

## Features
- Start screen with “Start Pokémon App” button
- Fetches all 151 Pokémon on load
- Dropdown with all 151 Pokémon names
- Displays selected Pokémon’s:
  -   Name
  -   Image
  -   Type(s)
  -   Weight
  -   Height

## Technologies Used
- React
- Fetch API
- PokéAPI: https://pokeapi.co/api/v2/pokemon?limit=151
